# StellarGov

> On-chain governance and treasury management protocol for DAOs, RWA funds,
> and enterprise multi-sigs on Stellar.

Stellar hosts $2B+ in real-world assets. Every product requires governance.
No open governance framework exists for Soroban. StellarGov is Compound Governor
for Stellar — proposal creation, on-chain voting, timelock execution, and treasury
management — built for the institutions already deploying on Stellar.
